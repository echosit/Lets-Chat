# Lets-Chat

A messaging web application. 

## Features

- Real-time users and messages
- Responsive UI Design
- User-generated chat rooms 
- Users in existing chat rooms are notified when other users enter or leave the chat

## Tools 

- JavaScript, HTML, CSS
- ReactJS
- NodeJS
- Socket.IO
